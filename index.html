jQuery Placeholder Fix - Test Case

Basic Usage

It will only apply the placeholder fix if the browser have no support to the placeholder attribute.

      $(document).ready(function(){
        $('form#basic').placeholder();
      });

Force placeholder

It will force the placeholder even if the browser have support.

      $(document).ready(function(){
        $('form#force').placeholder({
          force: true
        });
      });

Add styles to the placeholder

It will add the class placeholder to the fields. You can style your placeholder in the CSS file. 

      //javascript
      $(document).ready(function(){
        $('form#color').placeholder();
      });

      //css
      input.placeholder{
        color: #f00;
        font-style: italic;
      }
